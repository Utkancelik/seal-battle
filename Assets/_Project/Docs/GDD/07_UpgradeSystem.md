
### Upgrade Felsefesi

Upgrade’ler **istatistik artırmaz**, **oyun kurallarını değiştirir**.

#### Örnekler

- “Attack artık dururken çalışmaz”
    
- “Her düşman öldüğünde arena küçülür”
    
- “Hasar almazsın ama alan sürekli daralır”
    

### Data Yapısı

- ScriptableObject
    
- Gameplay sistemi sadece **interface** bilir
    
- UI sadece **description** okur
